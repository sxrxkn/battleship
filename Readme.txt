Расстановка кораблей искусственным интеллектом:
    Расстановка начинается с кораблей с наибольшим размером. Выбирается рандомная ячейка на поле, рандомно выбирается направление корабля.
    Вызывается функция проверки доступности ячеек, которые займет этот корабль, учитывая его направление и длину.
    Функция пробегается по всем ячейкам, начиная с переданной в функцию и проверяет на доступность.
    Данная операция повторяется, пока функция не вернет «true».

Ведение боя искусственным интеллектом: 
    Сначала бот стреляет по рандомным доступным ячейкам до попадания.
    Когда бот попадет, в store сохранится информация об успешном выстреле.
    Далее бот рандомно стреляет вокруг ячейки, в которую попал, до следующего успешного выстрела.
    Когда подбиты две ячейки и более, бот понимает направление корабля и будет стрелять только по нему, рандомно выбирая доступную сторону.
    После уничтожения корабля, алгоритм повторяется.
